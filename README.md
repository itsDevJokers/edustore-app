    - Node JS (Express)

    - Passport JS
    - CASL
    - Multer
- ### Database :
    - MongoDB
    - Mongoose


## ‚ñ∂Ô∏è Running the app

Clone the project

```bash
  git clone https://github.com/itsDevJokers/edustore-app.git
```

Go to the project directory & Install dependencies

- ### Client
```bash
  cd client
```
```bash
  npm install
```

- ### Server
```bash
  cd server
```
```bash
  npm install
```

Start the server & client concurrently

- Go into folder server and then 
```bash
  npm start
```
